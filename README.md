## 📖 О проекте

**GitSearch** — это полностью клиентское (Serverless / SPA) веб-приложение, созданное для удобного поиска репозиториев по всей базе GitHub. Оно предоставляет гораздо более гибкий интерфейс по сравнению со стандартным поиском, позволяя прямо «на лету» читать README, сохранять проекты в локальное избранное, переключать темы и фильтровать результаты без лишних перезагрузок страниц.

---

## 💡 Зачем это нужно?

1. **Экономия лимитов API:** По умолчанию неавторизованные запросы к API GitHub ограничены всего 60 обращениями в час. GitSearch объединяет локальную пагинацию с серверной, а встроенная авторизация через токен (Personal Access Token) снимает любые ограничения для комфортной работы.
2. **Абсолютная приватность:** Проект работает целиком в вашем браузере. Все ваши данные (избранное, токены авторизации, выбранная тема) хранятся исключительно локально в `LocalStorage` и никогда не передаются на сторонние бэкенд-серверы.
3. **Быстрый доступ к README:** Больше не нужно открывать десятки новых вкладок, чтобы понять, подходит ли вам проект. Описание каждого репозитория с гифками, изображениями и таблицами рендерится прямо внутри поисковой карточки по клику на кнопку «Подробнее».

## ✨ Ключевые возможности

* 🚀 **Умный поиск** — фильтрация по языкам программирования и сортировка по звездам, форкам или последним коммитам.
* 📂 **Оптимизированная пагинация** — запрашивает пакеты репозиториев по 100 штук за раз, но плавно выводит их порциями по 5, экономя ваш интернет-трафик и лимиты запросов.
* 📖 **Интерактивный README-вьюер** — загружает оригинальное описание прямо с серверов GitHub и на лету перезаписывает относительные пути картинок, гифок и ссылок на абсолютные (вы больше не увидите «битых» изображений).
* 🔖 **Локальное «Избранное»** — добавляйте проекты в закладки одним кликом, сортируйте их и храните локально на устройстве.
* 💾 **Экспорт и импорт** — сохраняйте базу своего избранного в файл `GS_export.json` и восстанавливайте её на любом другом устройстве.
* 🎨 **Кастомизация интерфейса**:
  * Три темы оформления: **Темная (по умолчанию)**, **Белая** и **Синяя**.
  * Возможность включения/выключения плавных анимаций интерфейса (для экономии батареи мобильных устройств).
  * Интерактивная пошаговая инструкция по получению токена внутри приложения.

## 🛠 Стек технологий

Приложение написано с упором на максимальную производительность без использования тяжелых фреймворков и сборщиков:

* **HTML5** — семантическая валидная разметка.
* **CSS3** — кастомные переменные (CSS Variables), адаптивная верстка (Flexbox, Media Queries), плавные переходы.
* **Vanilla JavaScript (ES6+)** — чистый код без внешних библиотек (зависимостей `0`).
* **GitHub REST API** — прямое взаимодействие для получения актуальных данных.
