Вход через GitHub Token
Как получить GitHub токен?
1. Войдите в свой аккаунт на сайте GitHub.
2. Откройте меню профиля справа сверху и выберите Settings.
3. В левой колонке в самом низу перейдите в Developer settings.
4. Откройте вкладку Personal access tokens -> Tokens (classic).
5. Нажмите кнопку Generate new token вверху страницы и выберите Generate new token (classic).
6. Укажите любое имя в поле Note и срок действия (Expiration).
7. Отмечать галочки (scopes) не нужно, если вы будете искать только публичные проекты.
8. Прокрутите вниз, нажмите Generate token и скопируйте созданную строку в это поле.
